MongoDB

Base de datos:
Mongodb trabaja con "colecciones" lo cual tiene adentro objetos de javascript (arrays o diccionarios en python).
Convierte los archivos JSON a BSON.

#Para pegar un texto copiado en mongodb se hace con el click derecho del mouse

Comandos:
db = Es para ver la base de datos actual que se esta utilizando (por defecto nos contecta a la base de datos test)
show dbs = Muestra todas las base de datos que tengamos disponibles
help = Muestra comandos que se pueden utilizar
use (nombre de base de datos) = de esta manera se crea una nueva base de datos. Ejemplo: "use webstore"
show collections = Sirve para mostrar las colecciones que contiene una base de datos
db.dropDatabase() = Sirve para eliminar una base de datos
db.createCollection("nombre") = Sirve para crear colecciones dentro de la db, se pasa el nombre como parametro entre comillas
db.coleccion.insert({"name": "laptop"}) = De esta manera se insertan valores dentro de mongo db "db.nombre.insert({"nombre": "valor"})
db.coleccion.drop = sirve para eliminar una coleccion que queramos, reemplazando coleccion por el nombre de la coleccion a eliminar
db.coleccion.find()= sirve para encontrar el contenido de la coleccion que le pasemos, podemos poner parametros dentro del parentesis para filtrar lo que queramos
db.coleccion.find().pretty() = Sirve para lo mismo que el de arriba pero devuelve el contenido de una manera mas visible
db.coleccion.findOne() = Sirve para buscar un solo dato, si hay varios datos con el mismo valor va a devolver solo el primero
db.colection.find().limit(2) = Sirve para limitar el resultado que nos otorga el .find()
db.colection.count() = Sirve para saber cuantos elementos hay dentro de la coleccion
db.colection.update({lo que se busca}, {lo que se agrega}) = Sirve para actualizar, lleva dos parametros necesarios, uno de busqueda y otro de actualizacion.
Quita todo el contenido previo y lo reemplaza por el segundo parametro otorgado
db.colection.update({lo que se busca},{$set: {lo que se va a agregar}}) = para agregar un valor se utiliza la update y se le otorga dos parametros de nuevo.
El primero de busqueda y en el segundo se agrega "$set" para que mongodb entienda que queremos agregar un valor seguido del valor que queremos agregar.
db.colection.update({busqueda},{$set:{lo que se agrega}},{upsert:true}) = Esto sirve cuando queremos "actualizar" un dato y de no existir que se agregue.
Para eso se agrega el "{upsert:true}"
db.colection.update({busqueda}, {$inc: {valor a incrementar: cuanto se va a incrementar}}) = Sirve para incrementar valores numericos dentro de la db
db.colection.update({busqueda},{$rename: {nombre de propiedad : nuevo nombre}} = Sirve para cambiarle el nombre a las "keys" dentro de los objetos
db.colection.remove({busqueda}) = Elimina el objeto que le pasemos en el parametro de busqueda, si no le ponemos ningun parametro va a eliminar todo lo que haya
